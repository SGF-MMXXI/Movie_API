Documentation Objectives

The purpose of this API is to build the server-side component of a Netflix type of application. The application will allow users to search through movie information

Example:

Allow new users to register /users POST A JSON object holding data about the new user
Example:
{ "username": "sammytest", "password": "1234", "email": "test@gmail.com", "Birthday": "01/01/1990" }
Allow users to update their user info /users/:Username PUT A JSON object holding data the user wishes to update with at least one of the following fields: FirstName, LastName, Username, Email, Password, Birthdate.
Example:
{ "FirstName" : "sammytest", "Password" : "1234" }
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST None A JSON object holding data about a user with an updated FavouriteMovies list
Example:
{ "_id": "618586c5ec86a4a75b2169a6", "FirstName": "sammy", "LastName": "test", "Username": "sammytest", "Email": "test@gmail.com", "Password": "1234", "Birthdate": "2004-04-05T00:00:00.000Z", "FavouriteMovies": [ "6182de309c3e0b19943474dc" ], "__v": 0 }

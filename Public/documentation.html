Documentation Objectives

The purpose of this API is to build the server-side component of a Netflix type of application. The application will allow users to search through movie information

REQUEST URL HTTP METHDOD REQUEST BODY DATA FORMAT RESPONSE BODY DATA FORMAT
Return a list of ALL movies to the user /movies GET None A JSON array holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:title GET None A JSON object holding data about a single movie, containing the title, genre, director and release year.
Return data about a genre (description) by name/title (e.g., “Documentary”) /movies/:title/genre GET None A text message indicating the genre(s) of a single movie
Allow new users to register /newUser POST A JSON object holding data about the new user
Example: { firstName: 'Ken', lastName: 'Washington' username: 'kdwiii', password: 'movies1', emailAddress: 'kenWashington@gmail.com' }
Allow users to update their user info (username) /newUser/:id/info PUT A JSON object holding data about the updated user information A text message indicating the user information was updated
Example: "Your username and password were successfuly updated."
Allow users to add a movie to their list of favorites /newUser/:id/favourites POST A JSON object holding data about a single movie, containing the title, genre, director and release year. A text message indicating the user successfully added a new movie to their favourites
Example: "Spider-man was successfully added to your favourites."
Allow users to remove a movie from their list of favorites /newUser/:id/favourites DELETE None A text message indicating the user successfully removed a movie from their favourites
Example: "Spider-man was successfully removed from your favourites."
Allow existing users to deregister /newUser DELETE None A text message indicating the user successfully removed a movie from their favourites
Example: "Your account was successfully deleted."
